# Random Quote Generator

**Автор:** [Надежда] [Портнова]

## Описание программы

Графическое приложение для генерации случайных цитат. Позволяет просматривать предустановленные цитаты, сохранять историю просмотров в файл JSON и фильтровать историю по автору или теме.

## Технологии

*   **Язык:** Python 3.x
*   **Библиотеки:** Tkinter (стандартная), json.

## Инструкция по запуску

1.  Установите Python (версия 3.6 или выше).
2.  Скачайте или клонируйте репозиторий.
3.  Запустите файл `quote_generator.py`:
    ```bash
    python quote_generator.py
    ```
4.  Приложение откроется в новом окне.

## Примеры использования и тесты

*   **Тест 1: Генерация цитаты**
    Нажмите кнопку «Сгенерировать цитату». Текст должен измениться на случайный из списка.
*   **Тест 2: История**
    Сгенерируйте несколько цитат. Проверьте список «История цитат» — он должен заполняться.
*   **Тест 3: Фильтрация**
    Введите имя автора (например, `Стив`) в поле фильтра и нажмите «Применить фильтр». В списке должны остаться только цитаты этого автора.
*   **Тест 4: Сохранение**
    Закройте приложение. Откройте файл `quotes.json`. Убедитесь, что массив `history` содержит последние просмотренные вами цитаты. Запустите приложение снова — история должна загрузиться.
